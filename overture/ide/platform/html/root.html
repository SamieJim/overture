Overture Tools for Formal Modelling in VDM

The Overture Platform

Overture is an open source tool for the Vienna Development Method (VDM). The tool supports different VDM dialects which include the ISO VDM-SL standard, as well as the object-oriented extension VDM++ and the real-time extension of that called VDM-RT.

Getting Started

  • See the help containing: the VDM Language Manual, the User Guide and a collection of examples. Go to 'Help' -> 'Help Contents' and look for Overture.
  • Visit the Overture website for general news about releases: OvertureTool.org
  • For help please use the "vdm++" tag on StackOverflow
  • Bugs should be reported on the GitHub tracker (registration required).

Additional Overture features

Overture offers additional tool features or plugins that have been developed in connection with other research projects. Currently, these features are:

  • The VDM2C code generator is developed as part of the INTO-CPS project, and enables code generation of VDM-RT models into C code.
  • The Overture FMU Simulation Exporter is an extension of Overture which enables VDM-RT models to be exported as Functional Mock-up Units (FMU) . Essentially, this feature enables co-simulation of VDM-RT and other formalisms that can be exported in FMUs. This plugin is also developed in the INTO-CPS project.
  • The Graphics Plotting enables plotting of VDM models, as they are being interpreted. Currently plotting is only supported for instance variables that contain numeric values, and which have been declared public and static. This project has been developed as part of the TEMPO project.

    • These plugins are available via updates sites and can be installed using Overture's update manager. This approach follows that of installing any other Eclipse plug-in: go to Overture's 'Help' menu, select the 'Install New Software . . . ' item and choose the plugin you want to install. For convenience, the update sites of the plugins described above are all included in Overture.
